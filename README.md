Controls:
  - draw:
    - left click: draw
    - right click: erase
  - colors:
    - r: switch to red
    - g: switch to green
    - b: switch to blue
    - n: switch to black
  - thickness (works with white color):
    - scroll up: increase thickness
    - scroll down: decrease thickness
  - file:
    - s: save
    - delete: clears the document
    - esc: escape
